因为学校又弄了新的平台（学校真有钱），所以被迫营业，现在支持智慧树平台了，欢迎使用。

说明：智慧树平台 web 端的前端是屎来着，都不知道为什么加载这么鬼慢，所以建议是有问题刷新几遍然后就好了。

## 文档

基于 Seleium 的刷课程序，用 Python 和 Selenium 实现。

运行前安装对应浏览器的 WebDriver（这里用的是 Edge），注意一下不同学校的登陆页不同，请自行修改。（如果你能够上到 GitHub 找到这个文件并跑起来，应该有能力自行修改）

## 配置文件说明

程序使用 `config.json` 文件来配置两个程序的网址和雨课堂的窗口数量。请根据需要修改该文件。

示例 `config.json`：

```
{
	"muke_url": "https://example.com/muke",  // 超星慕课的网址
	"yuketang_url": "https://example.com/yuketang",  // 雨课堂程序的网址
	"yuketang_window_count": 1  // 雨课堂窗口数量，用户可修改
}
```

请将网址替换为实际的网址，窗口数量根据需要调整。
