# NoMooc

基于 Seleium 的刷课程序，用 Python 和 Selenium 实现。

运行前安装对应浏览器的 WebDriver（这里用的是 Edge），注意一下不同学校的登陆页不同，请自行修改。（如果你能够上到 GitHub 找到这个文件并跑起来，应该有能力自行修改）

原理是先把每一个课程的链接都爬出来，然后打开每个链接，并点击开始学习，如果课程已经完成则跳过。

有问题请自己修改，你也可以提 issue。但是提 issue 的话我只有自己要刷课的时候才有机会改，因为没有测试的机会。如果你自己修改了，可以提 pull request 造福大家，不提也可以。

本仓库很不规范：没有 Release 版本，没有英文的 README，理由显而易见。

祝你大学生活愉快。

感谢使用。
